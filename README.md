HUES
=====

Gradient visualization animation.

# Get started

### Prerequirements
* [Node.js](https://nodejs.org)
* [Yarn](https://yarnpkg.com)

1. Install dependencies

```
yarn
```

2. Run dev environment

```
gulp
```

3. Run build

```
gulp build
```

4. Deploy to production

```
firebase deploy
```

5. Open production site https://hues-a6d41.firebaseapp.com/.
